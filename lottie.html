Lottie.js 官網
lottie動畫文件下載

這是一個簡單的動畫展示，使用 Lottie.js 加載 JSON 格式動畫。

範例程式碼

            
const animation = lottie.loadAnimation({
    container: document.getElementById('lottieAnimation'), // 動畫容器
    renderer: 'svg', // 渲染器類型 (svg, canvas, html)
    loop: true, // 是否循環播放
    autoplay: true, // 是否自動播放
    path: 'Animation - 1730289102299.json' // JSON 動畫文件的路徑
});
