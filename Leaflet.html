Leaflet.js 效果

Leaflet.js 官網

使用 Leaflet.js 嵌入地圖，可以顯示位置標記和路線。

範例程式碼

            
var map = L.map('map').setView([23.702211, 120.429469], 17); // 設置地圖中心點和縮放級別

L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', {
    maxZoom: 19,
    attribution: '© OpenStreetMap contributors'
}).addTo(map);

var marker = L.marker([23.702211, 120.429469]).addTo(map)
    .bindPopup('這裡是國立虎尾科技大學')
    .openPopup();
